MakuOS 10 Lite 1809 V4
(Windows 10 1809, 17763.6414)

Windows Defender удалён // Все обновления по 2024 год // Pro редакция // EN + RU + UA

Удалённые приложения:

Удалён Microsoft Edge, Windows Defender, Xbox и Microsoft Store.
Clipchamp, Microsoft Новости, Фотографии, Погода, Диктофон, Сообщения, Компоненты управления голосом, Календарь, Microsoft Solitaire Collection, Sticky Notes, 3D Viewer, Print 3D, OneDrive, Windows Hello, Windows Error Report, Feedback Hub, Get Help, Windows Sybsystem For Linux, Remote Assistance, Your Phone, Skype, Windows Mixed Reality, Easy Transfer.

Прочие изменения:

Интегрированы все обновления по конец октября 2024 года.
В сборке присутствует три языка: Английский, Русский, и Украинский.
Предустановлен MakuTweaker.
В сборке используется изменённое OOBE, в котором включён вход в локальную учётную запись, и скрыты какие либо страницы с рекламой.

Основное:

• Удалён центр обновления системы
• Включены скрытые файлы, папки, и диски
• Включены расширения файлов
• Откреплены закрепления на панели задач
• Отключён поиск Bing на панели задач

• Отключён SmartScreen
• Отключён быстрый запуск
• Отключена гибернация

• Отключена автоперезагрузка при синем экране смерти
• Включена иконка "Этот Компьютер" на рабочем столе
• Включён буфер обмена
• Убрана подпись "Ярлык" у новых ярлыков
• Отключено требование интернета в первоначальной настройке
• Разрешен вход в локальный аккаунт

• Включён старый просмотрщик фото
• Отключён Content Delivery Manager, который отвечает за предустановку рекламы
• Из панели задач убраны рекламные иконки
• Отключён SmartScreen и Включён DirectPlay
• Интегрированы NET Framework 2.0, 3.0, 3.5, и 4.7.2.
