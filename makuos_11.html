MakuOS 11 24H2
(Windows 11 24H2 26100.7015)

Windows Defender на выбор // Pro редакция // Русский + Английский + Украинский языки

Удалённые приложения:

Удалено отдельное приложение Cortana, Microsoft Карты, Windows Mixed Reality, Microsoft Новости, Погода, Диктофон, Советы, OneNote, Sticky Notes, Microsoft People, OneDrive, Почта, Feedback Hub, Календарь, Microsoft Solitaire Collection.

Изменения в сборке:

Отключена проверка на TPM 2.0, Процессор, и наличие Secure Boot.
На рабочем столе есть папка "от Марка Аддерли", в котором находятся установщики Google Chrome, 7-Zip, MPC-HC, Telegram, Zapret, Steam, Everything, а так же программа MakuYan которая запрещает Яндекс, ссылки на социальные сети, и Telegram канал с репаками без вирусов.

Там же имеется установщик MakuTweaker, которым можно применить различные твики, активировать систему, а так же очень эластично и подробно её настроить.

Применённые настройки:

• По умолчанию открывается "Этот ПК" вместо "Главная"
• Включены скрытые файлы, папки, и диски
• Включены расширения файлов
• Включена иконка "Этот Компьютер" на рабочем столе
• Отключена реклама и "предложения" в меню Пуск
• Отключено автошифрование диска с помощью Bitlocker
• Отключено залипание клавиш
• Отключена задержка контекстного меню
• Отключена автоперезагрузка при синем экране смерти
• Отключён Content Delivery Manager, который отвечает за предустановку рекламы
• Отключена различная телеметрия
